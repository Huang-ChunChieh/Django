Filter

Django Template Tag的Filter用於在實際返回變數前，先針對變數進行修改。在使用Filter的過程中，會很常看到|這個符號，可以把它想像成管道(pipe)，變數經過這個管道後，會產生不同的變化，像是將字串全部轉為大寫。

直接在變數中使用Filter

單一個Filter(|)

{% with firstname='Qi Xiang' %}

Hello {{ firstname|upper }}, how are you?

{% endwith %}

複數個Filter(|)

{% with firstname='Qi Xiang' %}

Hello {{ firstname|first|lower }}, how are you?

{% endwith %}

定義出使用Filter的區塊

單一個Filter(|)

{% filter upper %}

Hello everyone, how are you?

{% endfilter %}

複數個Filter(|)

{% filter upper|linenumbers %}my name is Emil.{% endfilter %} {% endblock %}
